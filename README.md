# IOC Extractor

A Python-based cybersecurity tool that extracts Indicators of Compromise (IOCs) from incident reports and threat intelligence data.

## Features

- Extract IP addresses
- Extract domains
- Extract URLs
- Extract email addresses
- Detect MD5 hashes
- Detect SHA1 hashes
- Detect SHA256 hashes
- Identify executable files

## Technologies

- Python 3
- Regular Expressions
- Kali Linux
- VS Code

## Sample Output

The tool automatically parses threat reports and extracts actionable indicators for investigation and threat hunting.
